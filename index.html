  • UFO Run is a fun game because it's very easy to learn how to play, and is simple as it only involves one control, the spacebar. The graphics are very nice and friendly. What's nice is that instead of the character dying upon first impact (like other similar games that I have played) the player is given a certain amount of hit points. That way you can have multiple collisions before you die. The only thing I don't like about the game is that the difficulty increases very quickly, so play time decreases.
